# `verify` 模板

## 目的

把验收标准转化为显式证据。

## 输入

- 已通过 `review` 的变更
- 验证契约
- 新鲜的检查结果

## 输出

- 验收点到证据的映射
- 检查状态
- 剩余风险
- `unverified` 或 `blocked` 项

## 通过条件

每个验收点都有直接证据，或被显式标记为 `blocked` / `unverified`。

## 来源约束

> 来源约束见 references/boundary-rules.md

## 模板

```md
## 验证

验收点 -> 证据:
-

证据来源:
-

验证方法:
-

已执行检查:
- 命令:
  结果:

未验证项:
- 

阻塞项:
- 

剩余风险:
- 

结论:
```
